The site you're looking at right now. Built with a specific philosophy: a portfolio should feel like the person who made it.

## Design Decisions

Monochrome with warm undertones — ink on cream, not sterile black on white. ASCII art as a first-class design element, not decoration. Typography does the heavy lifting: Departure Mono for code and structure, Instrument Serif for editorial headings, Literata for comfortable reading.

## The Dual-Mode System

The "I am human" / "I am agent" toggle isn't a gimmick. It's a statement about how content should adapt to its reader. Humans get editorial design, hover states, and visual hierarchy. AI agents get clean markdown, structured data, and semantic HTML. Same content, different rendering.

## Tech Stack

Full SSG via SvelteKit's static adapter. Zero runtime CSS frameworks. Content lives as markdown files processed by mdsvex. Total JavaScript: whatever Svelte needs and nothing more.
